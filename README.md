# NWordle

暑假期间，给自己和同学用 HTML+CSS+JavaScript 随手写的超轻量级的盗版 Wordle（一切灾难的源头：国家防火墙）。

玩法和原版 Wordle 一致，每天无限次试玩。单词表由 AI 生成。目前仅支持 PC 端使用浏览器游玩（才三天，而且白天还有网课，别那么着急嘛）。
